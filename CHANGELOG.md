## 版本规范

我们遵循 [语义化版本控制](https://semver.org/)：

- **主版本号** - 不兼容的 API 修改
- **次版本号** - 向下兼容的功能性新增
- **修订号** - 向下兼容的问题修正

## 更新类型

- `Added` - 新增功能
- `Changed` - 对现有功能的变更
- `Deprecated` - 即将移除的功能
- `Removed` - 已移除的功能
- `Fixed` - 任何 bug 的修复
- `Security` - 安全相关的修复
